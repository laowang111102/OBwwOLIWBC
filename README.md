# 智慧农场管理系统 java478

## 项目概述

智慧农场管理系统是一款集农田环境监测与农作物管理于一体的系统。该系统基于前后端分离的架构，后端采用 Springboot 与 Mybatis 框架，前端使用 Vue 框架配合 Element UI，以及 Ecahrts 进行数据可视化。

## 技术栈

- 后端：Springboot, Mybatis
- 前端：Vue, Element UI, Ecahrts

## 功能模块

### 用户管理

- 用户注册与登录
  - 支持：农场主、管理员
  - 功能：农场主注册与登录，验证码登录

### 角色权限管理

- 根据不同角色分配访问权限，保护系统安全。

### 作物管理

- 作物信息管理
  - 功能：添加、编辑、删除作物信息，包括作物名称、类型、种植时间、预计收成时间、种植面积、种植环境，显示作物图片

### 生长记录

- 记录作物的生长状态、施肥和灌溉情况，包括周数、施肥数量、灌溉次数、日照时长、增长高度与宽度、病虫情况、叶面状态、土壤湿度等

### 收成管理

- 收成记录
  - 记录每个作物的实际收成量、收成时间及相关备注

### 销售管理

- 市场销售记录
  - 功能：登记销售的作物类型、数量、客户信息

### 统计分析

- 对收成数据进行统计，生成柱状图报告以便分析和决策

### 收益分析

- 计算每种作物的收益，用柱状图统计，并生成报表

### 农业问答

- 功能：提供常见问题解答、种植技巧和病虫害防治等信息，支持问题与答案的录入，搜索权重排序

### 管理员界面

- 发布化肥、农药、农具产品广告
  - 在首页以滚动条形式显示管理员发布的产品广告
